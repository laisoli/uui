EXTRUDING A 3D MODEL FROM A 2D IMAGE

A 3D model can be created from a 2D image using the image import feature in Cura. Cura can import .JPG .PNG .BMP and .GIF file types.

The minimum and maximum heights of the image can be set by the user, then Cura uses the grayscale of the image to set the height of different areas of the image (within the minimum and maximum dimensions).

Importing an image into Cura

Open Cura and in the top menu, select File, then Open File(s). Browse to the image you wish to import, select it then click the Open button. The Convert Image window is then shown.

Set the required Height, this is the maximum extruded height of the print in millimetres.

Set the required Base, this is the base height of the print in millimetres, set this to 0.0 if you do not want to print a base.

Set the required Width or Depth of the print in millimetres. When one of these dimensions is entered, the other is calculated in proportion to the size of the original image.

Select Darker is higher or Lighter is higher. This selection will make either the darker areas of the image print higher or the lighter areas of the image to print higher. For the settings shown in the example above, the black areas of the image will print 10mm high and the white areas of the image will print at 1mm high.

 Greyscale will print in between these heights dependent upon the percentage of greyscale, so 50% grey would print at a height of 5.5mm. Set the required Width or Depth of the print in millimetres. When one of these dimensions is entered, the other is calculated in proportion to the size of the original image.

 Drag the Smoothing slider to select how much smoothing to apply between the colour transitions.

Click OK, this will import the image into Cura and convert it into a 3D model with the selected settings. The 3D model can then be manipulated in Cura and prepared for 3D printing in the same way as standard 3D models.
